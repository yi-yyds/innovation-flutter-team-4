## 📦 快速开始

### 环境要求
- Flutter SDK 3.0 或更高版本
- Dart SDK 3.0 或更高版本
- Android Studio / VS Code (推荐)

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/yi-yyds/change_flutter.git

# 进入项目目录
cd change_flutter

# 安装依赖
flutter pub get

# 运行项目
flutter run
```

### 构建命令

```bash
# Android APK
flutter build apk --release

# iOS (需要 macOS)
flutter build ios --release

# Web
flutter build web

# Windows
flutter build windows
```
